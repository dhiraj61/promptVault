# promptVault
PromptVault is full-Stack MERN application that allows users to create,manage, and explore AI prompts.
Each prompt is automatically tagged using Gemini AI, making it easier to discover through intelligent search.
Users can maintain personal collections or shape prompts publicly fot the community to like, copy or reuse.

Features:
Secure Authentication (JWT + Cookies)
Create, Edit, and Delete Prompts
AI-Powered Tag Generation (Gemini API)
Public and Private Vaults
Search by Tag or Keyword
Like & Explore Prompts for Others
